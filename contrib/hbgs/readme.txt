Name:                  Ghostscript [multiplatform, free, open-source]
URL:                   http://ghostscript.com/
                       http://pages.cs.wisc.edu/~ghost/
Environment variable:  HB_WITH_GS=C:\ghostscript\psi
                       HB_WITH_GS_BIN=C:\ghostscript\bin (on Windows)
Install (debian):      sudo apt-get install libgs-dev
Install (rpm):         ghostscript-devel
Install (OS X):        brew install ghostscript
