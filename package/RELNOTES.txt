Release Notes
=============

Windows release for x86 and x64 platforms.

Default components: x86 tools, x86 static/shared libs, examples.
Options: MinGW compiler, x64 libs and tools.

Usage:
1. install/unpack to any directory
2. click "Harbour Shell" on your Desktop
3. install MinGW compiler by typing: `getmingw.bat`
4. for x86 executable, type: `hbmk2 ../tests/hello.prg`
5. for x64 executable, type: `hbmk2 ../tests/hello.prg -cpu=x64`
